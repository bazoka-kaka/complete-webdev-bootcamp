BOOTSTRAP CARDS
===============

- you just have to copy it from bootstrap and customize it 

- LESS TIME SAME RESULT MORE MONEY!

here's the example:
https://getbootstrap.com/docs/4.1/examples/pricing/

just go to the source, then copy the part 
that you wanna use then change it as your liking

another source:
https://bootsnipp.com/

here's the docs for bootstrap card:
https://getbootstrap.com/docs/4.0/components/card/

note:
bootstrap card is just a container of card 

- image
- description 
- button 

CREATING CARD FROM SCRATCH
--------------------------

<div class='card'>
    <div class='card-header'>
        card header
    </div>
    <div class='card-body'>
        card body
    </div>
    <div class='card-footer'>
        card footer
    </div>
</div>

note:
a card has 3 components, header, body, and footer, 
or just the things that you wanna include

  STYLING OUR CARD
  ----------------

  - insert all your card to a parent div with the class 
  "card deck"

  note:
  card deck makes the card inline, the cards get distributed 
  evenly accross the width of the website, they also end up 
  with the same height 

  one problem:
  not as adaptable as the bootstrap grid system 

  - challenge:
  using the bootstrap grid system, make as such:
  1. on the largest screen, make the cards all inline 
  2. but on the tablet make two cards inline, and leave the last 
  one fill the entire width 
  3. on phone, make the entire card inline in a col 

  buttons
  -------
buttons property for our cards:
class="btn btn-lg btn-block btn-outline-primary"

note:
btn-block -- it's gonna take the full width of our 
card
btn-lg -- the size of the button 
btn-outline-primary -- this is gonna be the outlined button style  
btn-primary -- this is gonna be the button style 
(not outlined style)
