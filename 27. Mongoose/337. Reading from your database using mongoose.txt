READING FROM YOUR DATABASE USING MONGOOSE 
=========================================

- to read use the following:
Fruit.find(function (err, fruits) {
  if (err) {
    console.log(err);
  } else {
    console.log(fruits);
  }
});

- challenge:
loop trough the array of fruits and only log their name 
using forEach 

solution:
Fruit.find(function (err, fruits) {
  if (err) {
    console.log(err);
  } else {
    fruits.forEach((fruit) => {
      console.log(fruit.name);
    });
  }
});

note:
to close the mongoose after we got what we want:
mongoose.connection.close()

application:
Fruit.find(function (err, fruits) {
  if (err) {
    console.log(err);
  } else {
    mongoose.connection.close();
    fruits.forEach(function (fruit) {
      console.log(fruit.name);
    });
  }
});
