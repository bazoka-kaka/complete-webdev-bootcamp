MONGODB CRUD OPERATIONS IN THE SHELL: UPDATE 
============================================

- updating data in our database by adding 
a stock value to each of our documents 

>> db.products.updateOne({_id: 1}, {$set: { stock: 32}})
note:
this is gonna update the data which has an id 
of 1 and set the stock to 32 

challenge:
update the document that has an id of 2 
to have a stock field which has the value of 12 

answer:
>> db.products.updateOne({_id: 2}, {$set: {stock: 12}})
